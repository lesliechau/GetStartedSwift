Add an Automated DevOps Toolchain

Add a DevOps Toolchain

In this step you will add an automated DevOps pipeline using the Bluemix DevOps Toolchain. Each Starter includes a toolchain.yml file that automatically configures a DevOps toolchain in Bluemix.

Initialize a new github repository for your project using the following commands in your terminal or command prompt:

git init
git remote add origin <repo>
git add .
git commit -m "My Project"
git push

Open a web browser and navigate to your github repository:

'http://github.com//

Next, click on the "Create Toolchain" button to initilize the process. This brings you to Bluemix where you need to perform a manual step to verify your configuration and trigger the creation of the toolchain.

Each Starter project provides a recommended toolchain configuration for an automated build, deploy and delivery process. Learn more about DevOps toolchain.
